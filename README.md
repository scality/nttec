# Build from source

## Build

```sh
mkdir build
cd build
cmake -G 'Unix Makefiles' ..
make
```

### Targets

- `ntl_bench`: build the NTL benchmark
- `ntl_shared`: build the NTL shared library
- `ntl_static`: build the NTL static library
- `tools`: build the NTL tools
- `benchmark`: run the NTL benchmark
